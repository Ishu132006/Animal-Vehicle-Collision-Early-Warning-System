# 🦌 AI-Based Animal–Vehicle Collision Detection and Early Warning System

## 📌 Project Overview

This project presents an AI-powered road safety solution designed to detect animals near roadways and provide early warnings to drivers before potential collisions occur.

Using YOLOv8n, Computer Vision, and Deep Learning techniques, the system performs real-time animal detection, collision risk assessment, danger zone monitoring, and audio-visual alert generation to improve road safety and wildlife protection.

---

## 🚀 Key Features

- Real-Time Animal Detection using YOLOv8n
- Collision Risk Classification (Low, Medium, High)
- Audio Warning Alerts
- Visual Warning System
- Real-Time Dashboard Display
- Collision Prediction Module
- Automatic Alert Logging
- Danger Zone Visualization
- Efficient CPU-Based Deployment

## 📊 Dataset Information

The model was trained using a combined animal image dataset prepared for object detection tasks.

### Dataset Statistics
- Training Images: 3,561
- Validation Images: 538
- Image Resolution: 640 × 640
- Class: Animal

---

## 🔄 System Workflow

1. Capture video frames from road cameras or dashcam footage
2. Detect animals using YOLOv8n
3. Analyze object position and movement
4. Assess collision risk level
5. Generate visual and audio alerts
6. Log high-risk events automatically
